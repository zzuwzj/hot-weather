# hot-weather
React SSR demo with Next.js

## steps:
- clone this project to your local sys
- cd to project folder
- install dependencies
```shell
npm i
```
- start
```shell
npm start
```
- open browser and visit home page:  
http://127.0.0.1:3000/  
or detail page:  
http://127.0.0.1:3000/detail

### enjoy it :)
